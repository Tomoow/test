# Design System

A modern design system component library built with React, TypeScript, and Storybook.

## Components

- **Button** - Primary button component with variants and sizes

## Development

### Install dependencies

```bash
npm install
```

### Run Storybook locally

```bash
npm run storybook
```

### Build Storybook

```bash
npm run build-storybook
```

## Storybook

The design system is published and hosted on GitHub Pages. View it at:

**https://tomoow.github.io/test/**

## Design Tokens

All design tokens are semantic and located in:
- `src/styles/tokens.css` - CSS variables
- `src/tokens/index.ts` - TypeScript tokens

## Contributing

1. Make changes to components
2. Test in Storybook locally
3. Commit and push changes
4. GitHub Actions will automatically deploy to GitHub Pages
